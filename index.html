// Getting Started

Cách tham gia máy chủ

01
Mở Minecraft

Khởi động phiên bản Minecraft Java Edition trên máy tính (PC) hoặc Bedrock Edition trên điện thoại (Mobile) của bạn.

02
Thêm Máy Chủ

Vào mục Multiplayer (Chơi mạng) → Chọn Add Server (Thêm máy chủ) → Nhập chính xác địa chỉ IP của máy chủ.

03
Kết Nối & Chơi

Bấm nút Join Server (Tham gia máy chủ), và bắt đầu cuộc hành trình phiêu lưu của bạn!
